Privacy Policy

Last updated: April 2026

🔒 The Short Version

DevMind processes everything locally on your device. No data is sent to external servers. No tracking. No analytics. No telemetry. Your conversations stay on your machine.

1. Data Collection

DevMind does not collect, transmit, or store any personal data on remote servers. Specifically:

  • No analytics — We do not use Google Analytics, Mixpanel, or any tracking service.
  • No telemetry — No usage metrics are sent anywhere.
  • No cookies — DevMind does not set or read cookies.
  • No third-party AI APIs — All AI processing runs locally via Chrome AI (Gemini Nano) or your own Ollama server.

2. Data Stored Locally

DevMind stores the following data locally in your browser using the Chrome Storage API:

  • Settings — Your preferences (Ollama URL, model, theme, etc.) stored in chrome.storage.sync.
  • Conversation History — Your chat messages per website, stored in chrome.storage.local. You can clear this at any time from Settings.
  • Usage Stats — Simple counters (total requests, session requests) stored locally. Never transmitted.
  • License Key — If you purchase Pro, your license key is stored locally for validation.

All locally stored data can be deleted at any time by clearing your browser data or uninstalling the extension.

3. AI Processing

DevMind uses two AI engines, both of which run entirely on your device:

  • Chrome AI Built-in (Gemini Nano) — Runs natively inside Chrome. No network requests.
  • Ollama — Connects to a local server (default: localhost:11434) that you install and control. Requests never leave your machine.

No prompts, responses, or page content are ever sent to cloud AI providers (OpenAI, Google Cloud, Anthropic, etc.).

4. Permissions

DevMind requests the following browser permissions:

  • storage — To save your settings and conversation history locally.
  • scripting — To inject the assistant panel into web pages.
  • contextMenus — To add right-click menu actions (translate, summarize, debug, etc.).
  • Host permissions (all URLs) — Required to inject the content script on all pages and to connect to your local Ollama server at localhost:11434.

5. License Validation

If you purchase DevMind Pro, the extension will make a single HTTPS request to the Gumroad API (api.gumroad.com) to validate your license key. This request contains only your license key — no personal data, browsing data, or conversation content is included.

If validation fails (e.g., you're offline), the extension uses optimistic activation and will retry when connectivity is restored.

6. Third-Party Services

DevMind uses the following external services only when explicitly triggered by you:

  • Google Fonts — Typography loaded from fonts.googleapis.com for the extension UI.
  • Gumroad — License validation for Pro purchases (only when you enter a license key).

7. Children's Privacy

DevMind does not knowingly collect data from children under 13. Since no personal data is collected at all, this policy applies universally.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last updated" date above and distributed via extension updates on the Chrome Web Store.

9. Contact

If you have any questions about this Privacy Policy, please reach out via the Chrome Web Store listing or our GitHub repository.
